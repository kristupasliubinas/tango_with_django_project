Building Rango to learn Django
--------------

A web application developed for the purpose of learning Django framework. It lets people browse user-defined categories of website links (similar to a book index). The application is called Rango and is taught in the book 'Tango with Django' by Leif Azzopardi and David Maxwell.

Project requirements:
- Python 2.7
- Django 1.9/1.10/1.11
